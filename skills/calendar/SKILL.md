# Calendar Skill

Gestion de l'agenda d'Omar via Google Calendar API.

## Opérations
- Lister les RDV du jour/semaine
- Créer un rendez-vous
- Modifier/annuler un RDV
- Rappels automatiques (1h avant)
- Bloquer des créneaux (focus time)

## Préférences Omar
- RDV clients : matin (9h-12h)
- Focus/admin : après-midi (14h-17h)
- Pas de RDV le vendredi après-midi
- Train : 1ère classe Part-Dieu, Carte Avantage
- Courses : Carrefour Villeurbanne 18h-20h
